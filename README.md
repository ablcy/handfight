# 手势识别剪刀石头布

一款纯前端实现的手势识别剪刀石头布游戏，使用 Google MediaPipe Hands 模型进行手势识别。

## 功能特点

### 🎮 核心功能
- 📷 摄像头实时手势识别
- ✊ 识别石头、剪刀、布三种手势
- 🤖 自动与 AI 对战
- 📊 实时计分系统
- 🎚️ 可自定义 AI 胜率

### 🎨 界面布局
- 🎯 左上角：AI 手势区（1/4）
- 🎯 右上角：你的手势区（1/4）
- 🎯 下半部分：裁判区（1/2）
- 📱 响应式设计，支持电脑和手机

### 🕵️ 隐私保护
- 🎚️ 可隐藏摄像头画面
- 隐藏后仍可正常识别手势
- i人福音，避免尴尬！

### ⚙️ 设置功能
- 🔧 可调节 AI 胜率（0%-100%）
- 0% = 你每局必赢
- 100% = 你每局必输
- 🔄 一键重置分数

### 📋 更新日志
- 页面底部显示完整更新记录
- 默认折叠，节省空间
- 最新内容在最上面

## 技术栈

- 🌐 HTML5
- 🎨 CSS3
- 💻 JavaScript
- 🤖 Google MediaPipe Hands

## 使用说明

### 🚀 快速开始
1. 在浏览器中打开 `index.html`
2. 允许访问摄像头
3. 将手放入摄像头画面
4. 做出石头、剪刀或布的手势
5. 开始对战！

### 🎯 手势说明
- ✊ 石头：握拳
- ✌️ 剪刀：伸出食指和中指
- 🖐️ 布：张开手掌

### ⚙️ 设置说明
1. 点击「设置」按钮
2. 拖动滑块调节 AI 胜率
3. 可选择隐藏摄像头
4. 点击「重置分数」清空计分板

### 🎮 游戏规则
- 石头赢剪刀
- 剪刀赢布
- 布赢石头
- 相同手势平局
