# 手势识别 - 剪刀石头布对战

一款纯前端实现的摄像头手势识别剪刀石头布对战游戏，无需后端和数据库，打开即可使用！

## 功能特性

### 核心功能
- 🎥 摄像头实时手势识别
- 🤖 AI自动对战
- 📊 实时计分系统
- ⚙️ 自定义AI胜率调节

### 手势识别
- 石头（✊）：所有手指收起
- 剪刀（✌️）：食指和中指伸出
- 布（🖐️）：所有手指伸出

### 胜率调节
- **0%**：你每局必赢
- **100%**：你每局必输
- **中间值**：按对应概率随机对战

## 使用说明

### 1. 打开游戏
直接用浏览器打开 `index.html` 文件即可

### 2. 授予权限
首次打开时，浏览器会请求摄像头权限，请点击"允许"

### 3. 开始游戏
将手放入摄像头视野，做出石头、剪刀或布的手势
- 手势需要保持稳定一段时间才能触发对战

### 4. 调节胜率
点击"⚙️ 胜率设置"按钮展开设置面板
- 拖动滑块或直接输入数值调节AI胜率
- 设置实时生效，无需刷新页面

### 5. 重置分数
点击设置面板中的"🔄 重置分数"按钮可清空所有计分

## 技术实现

- **手势识别**：Google MediaPipe Hands
- **前端框架**：原生HTML + CSS + JavaScript
- **视觉效果**：渐变色、响应式布局、动画效果
